☁️ Cloud Sync

Sync across Mac & iPhone via JSONBin.io (free)

JSONBin.io stores your data as a JSON file in the cloud, accessible from any device. Setup takes ~2 minutes:

  1. Go to jsonbin.io → sign up for a free account
  2. Click + New Bin → paste the following as content → Save:
    {"version":2,"sessions":[],"nextdays":[],"routines":[]}
  3. Copy the Bin ID from the URL (the part after /b/)
  4. Go to Account → API Keys → copy your Master Key
  5. Paste both below and click Connect
